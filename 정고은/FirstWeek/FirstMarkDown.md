## 글머리를 넣는 방법

```
# == H1, ## == H2 등등
```

>  인용 문자를 넣는 방법

```
> 어쩌구저쩌구
```

> 안에 
>
> * 여러가지
> * - 넣을 수 있다.

1. 순서는
2. 그냥
3. 쓴다.



글머리 기호는 *, +, - 이다.

* 일번

+ + 이번
  + - 삼번



코드블럭은 ``` 세개 쓰기

```
그것만 기억하자구... 어려우니께
아니면
<pre>
<code
어쩌구
</code>
</pre> 처럼 할 수 있다 하지만 코드블럭코드가 좋은거같다.
```

수평선을 쓰는 법 <hr/>

```
